# jacksangster03.github.io

Personal portfolio site. Built with plain HTML and CSS, hosted on GitHub Pages.

**Live:** [jacksangster03.github.io](https://jacksangster03.github.io)

## About

Biochemistry graduate from Imperial College London, currently studying Management and Computer Science & Business Technology at IE University in Madrid.

The site covers projects, skills, and background. No frameworks, no build step.

## Stack

- HTML5 + CSS3 (custom properties, CSS Grid, Flexbox)
- [DM Serif Display](https://fonts.google.com/specimen/DM+Serif+Display) + [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- Hosted on GitHub Pages

## Structure

```
index.html   — single-page site
style.css    — all styles
```

## Local preview

Open `index.html` directly in a browser. No build process required.
